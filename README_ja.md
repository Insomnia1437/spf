# 簡易ポート転送ツール (Simple Port Forwarder - SPF)

[English](README.md) | [简体中文](README_zh.md) | [日本語](README_ja.md)

SPF は、SSH ポートフォワーディング（ローカル、リモート、ダイナミック転送）の管理・実行を行うための、軽量でポータブルな Windows 専用ツールです。

![Simple Port Forwarder GUI](docs/gui.png)

## 主な機能

- **ポートフォワーディングモード**: ローカルポート転送 (`-L`)、リモートポート転送 (`-R`)、およびダイナミックポート転送 (`-D` SOCKS5 プロキシ) をサポート。
- **安全性優先の設計**: SSH パスワードは接続実行時のみメモリ上に入力され、設定ファイルには一切保存されません。
- **ポート競合の検出**: 接続開始前にローカルポートが使用中であるか自動でチェックし、競合エラーを未然に防止。
- **自動再接続**: ネットワーク切断を検知した場合、Exponential Backoff（指数バックオフ）アルゴリズムを用いて自動的に再接続を試行。
- **ログコンソール**: 接続ログ、データフローの詳細、および例外エラーをリアルタイムで分かりやすく表示。
- **設定ファイルのインポート・エクスポート**: 全設定のバックアップ・共有が容易。既存トンネルのコピー・複製機能も搭載。

## 使用方法

**GitHub Releases** ページからビルド済みのバイナリをダウンロードしてください。

ソースコードからビルドした場合、バイナリは `publish/` ディレクトリ内に生成されます：
- `publish/SPF.exe` (軽量依存版: 動作には .NET 8.0 Desktop Runtime のインストールが必要です)
- `publish/self_contained/SPF.exe` (自己完結版: 必要なランタイムが同梱されているため、そのまま単体で動作します)

## 開発とビルド

.NET 8.0 SDK のインストールが必要です。以下のコマンドでビルドが実行できます：
```bash
dotnet build SPF.sln
```
